# Competitor Scan Sub-Prompt

Given the scraped competitor pages in the artifact (Postery, ArtPix, Kraftly, plus any others in `focus.md`):

1. **New arrivals**: What did each competitor add in the last 14 days? Which of those overlap our categories?
2. **Price moves**: Any category-level price changes vs our snapshot? Are we above or below?
3. **Copy patterns**: What phrasing appears on 2+ competitors that we're missing? (e.g. "handcrafted in India", "museum-grade paper".)
4. **SEO gaps**: Any keyword the competitor now ranks for that we don't have a page for? (Cross-reference with GSC content gaps.)
5. **Recommendations**: Top 3 tactical moves for the next 14 days. Each move = one candidate PR OR one design brief for a new SKU.

Never copy competitor copy verbatim. Extract the pattern, rewrite in our voice.
